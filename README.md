# fifa-world-cup
Predicting the FIFA World Cup 2026 using historical tournament data, machine learning, time series analysis, and Monte Carlo simulation. This project analyzes nearly a century of FIFA World Cup history to estimate team performance and championship probabilities through statistical modeling and predictive analytics.
The FIFA World Cup is one of the world's most competitive sporting events, where historical performance, current team strength, and tournament dynamics all influence the outcome. The objective of this project is to investigate whether historical data and modern machine learning techniques can be used to forecast the 2026 FIFA World Cup.
Using nearly a century of World Cup data, this project combines historical match results, tournament qualification records, and FIFA world rankings to build predictive models capable of estimating team performance throughout the tournament. Multiple machine learning algorithms were evaluated alongside Monte Carlo simulations to estimate the probability of teams advancing through each stage and ultimately winning the championship.
Rather than attempting to predict a single tournament bracket, this project focuses on probabilistic forecasting by simulating the tournament thousands of times to quantify uncertainty and compare competing teams.
